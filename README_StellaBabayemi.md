# NUMBER GUESSING GAME

## ALGORITHM

1. Start
2. Set secret = 5
3. Set guess = 1-10
4. If guess = secret
5.    Print "Correct"
6. Else If guess < secret
7.     Print "Too low"
8. Else
9.    Print "Too high"
10. End
